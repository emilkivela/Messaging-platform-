# Messaging-platform-
Keskustelusovellus, jonka avulla voi liittyä tekstikanaville tai luoda oman tekstikanavan. Sovelluksen tarkoituksena on toimia alustana esimerkiksi ryhmäkeskusteluna kaveriporukalle, keskustelufoorumina liittyen johonkin tiettyyn aiheeseen, tai vaikkapa pelialustana mafiosolle tai muille vastaaville rymäpeleille.

Sovelluksen ominaisuuksina voit liittyä useille kanaville, luoda omia keskusteluja sekä hallinoida omia kanaviasi poistamalla viestejä ja vaihtamalla käyttäjien nimiä kyseisellä kanavalla. Käyttäjät voivat etsiä aivainsanan sisältävän viestin, ja he näkevät kuinka monta viestiä kullakin kanavalla on lähetetty sekä milloin mikäkin viesti on lähetetty.
